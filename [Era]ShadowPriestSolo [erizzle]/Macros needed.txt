# Instructions to Create the "Shoot" Macro in World of Warcraft
# This is Mage/Priest only macro
1. Open the in-game macro interface by typing `/macro` in the chat.
2. Click on "New" to create a new macro.
3. Enter the name of the macro: `Shoot`
4. Choose an icon for the macro (optional).
5. Copy the following macro code and paste it into the macro text box:

		/cast !Shoot

6. Click "Save" to save the macro.
7. Exit the macro interface.
8. Type `/reload` in the chat to reload the UI.
